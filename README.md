# JaruratCare Mini Healthcare Support App

## Overview
JaruratCare is a concept-level mini healthcare support web app built for NGO or volunteer use. It helps collect patient support requests through a simple form and includes a basic AI/automation feature to summarize the request and detect urgency.

## Features
- Patient help request form
- Automatic request summary
- Priority detection (High / Medium / Low)
- Recent request display
- Clean and responsive user interface

## AI / Automation Idea
The app uses keyword-based logic to analyze the user's problem description. Based on the text entered, it:
- generates a short summary
- detects urgency level automatically

This helps NGOs or volunteers quickly identify which requests need faster attention.

## NGO Use-Case
This app can help an NGO:
- collect patient requests in a simple digital format
- reduce manual reading effort
- prioritize emergency cases first
- organize healthcare support more efficiently
