Why this fits the NGO use-case

The app is designed for NGOs, volunteers, or community healthcare teams that need to quickly understand incoming requests and decide which case needs immediate attention.

🤖 AI / Automation Idea Keyword-based urgency detection and instant text summarization for patient support requests.
⚡ Practical Value Reduces manual effort in reading long messages and helps volunteers focus first on critical cases.
📦 Data Handling Submitted requests are stored locally in the browser using localStorage for demo purposes.
Built as a concept-level mini healthcare support prototype for internship evaluation.
